Get Started with n4s

Select your operating system for step-by-step instructions.

Guide for Windows

  1. Download and install Docker Desktop from Docker Desktop for Windows.
  2. Start Docker Desktop and ensure it's running properly.
  3. Open a PowerShell terminal and pull the n4s Docker image: 
    docker pull n4s/accounting
  4. Run the container: 
    docker run -it --name n4s-container n4s/accounting
  5. Interact with the solution in the terminal.

Guide for macOS

  1. Download and install Docker Desktop from Docker Desktop for macOS.
  2. Start Docker Desktop and ensure it's running properly.
  3. Open Terminal and pull the n4s Docker image: 
    docker pull n4s/accounting
  4. Run the container: 
    docker run -it --name n4s-container n4s/accounting

Guide for Linux

  1. Install Docker by running: 
    sudo apt update && sudo apt install -y docker.io
  2. Start the Docker service: 
    sudo systemctl start docker
  3. Check that Docker is running: 
    docker --version
  4. Pull the n4s Docker image: 
    docker pull n4s/accounting
  5. Run the container: 
    docker run -it --name n4s-container n4s/accounting
